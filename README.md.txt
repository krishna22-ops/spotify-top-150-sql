# Spotify's Top 150 Track Analysis - SQL Data Analysis

## Short Description  
This project analyzes Spotify's Top 150 tracks using MySQL. It demonstrates database design, data normalization, and writing SQL queries to answer real-world music-related questions. The dataset includes artist, track, genre, and streaming information.

## Folder Structure  
- **Schema** – SQL script to create the database schema  
- **Insert Data** – SQL script to populate tables with sample data  
- **Queries** – Analytical queries for reports and insights  
- **ERD.png** – Entity Relationship Diagram of the database  
- **README.md** – This file

## Technologies Used  
- MySQL Workbench

## Source  
[Most Streamed Spotify Songs 2024 - Kaggle](https://www.kaggle.com/datasets/nelgiriyewithana/most-streamed-spotify-songs-2024)

## Sample Queries  
- Tracks with over 200M streams released before 2020  
- Top 3 genres by total streams  
- Artist with the most total streams  
- Rank each artist's tracks by stream count
